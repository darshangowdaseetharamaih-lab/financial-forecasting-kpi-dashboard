# Financial Forecasting & KPI Dashboard

## Project Overview
Developed an end-to-end financial analytics solution to track revenue, expenses, liquidity, and key performance indicators (KPIs).

## Business Problem
Finance teams require timely visibility into financial performance to support data-driven decision-making and forecasting.

## Solution
Built a cloud-enabled analytics workflow using structured financial data, SQL-based analysis, and interactive dashboards.

## KPIs Tracked
- Revenue & Revenue Growth
- Operating Expenses
- Net Profit & Profit Margin
- Cash Balance
- Liquidity Ratio
- Forecast vs Actual Variance

## Tools & Technologies
- AWS S3
- AWS Athena
- SQL
- Power BI
- Excel (Power Query)
